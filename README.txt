Мой школьный проект в 10-ом классе:
    Основные технологии которые использовал:
        - Django
        - sqlite3
        - requests
        - Pillow
        - beautifulsoup4
    В проекте работает все, основные элементы
        - сайт и все фильтры и поиск на нем
        - админ панель (например редактирование базы данных и другое, лучше зайти и посмотреть)
        - база данных
        - Парсер (собирает полную информацию о задачах в Базу Данных с сайта: https://www.problems.ru/)


Запуск проекта:
    1) перед началом установить библиотеки из requirements.txt
    2) перейти в директорию "Olympiadnik\mysite"
    3) написать команду: 'python manage.py runserver' в терминале и сайт откроется на localhost

Переход в админ панель:
    Перейти в браузере: http://127.0.0.1:8000/admin/
    Логин: admin
    Пароль: admin
